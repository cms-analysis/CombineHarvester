1 Post-fit Distributions {#post-fit-shapes}
2 =========================================
3 
4 **File**: CombineTools/test/PostFitShapes.cpp
5 
6 The program `PostFitShapes` is used to dump a set of pre- and post-fit distributions from a datacard. There is no need to specify which categories or processes to extract, this will be done automatically and so should work for any shape-based analysis. The effect of all normalisation and shape uncertainties is evaluated. In addition, there is an option to calculate post-fit/pre-fit scale factors for each background and the total background uncertainty in each category. This numbers can then be used as input for producing post-fit control distributions. The following options are available:
7 
8  Configuration:
9  -d [ --datacard ] arg The datacard .txt file [REQUIRED]
10  -o [ --output ] arg Name of the output root file to create [REQUIRED]
11  -f [ --fitresult ] arg Path to a RooFitResult, only needed for postfit
12  -m [ --mass ] arg Signal mass point of the input datacard
13  --postfit [=arg(=1)] (=0) Create post-fit histograms in addition to pre-fit
14  --sampling [=arg(=1)] (=0) Use the cov. matrix sampling method for the
15  post-fit uncertainty
16  --samples arg (=0) Number of samples to make in each evaluate call
17  --print [=arg(=1)] (=0) Print tables of background shifts and relative
18  uncertainties
19  Example usage:
20  PostFitShapes -d htt_mt_125.txt -o htt_mt_125_shapes.root -m 125 \
21  -f mlfit.root:fit_s --postfit --sampling --print
22 
23 \warning The option `--sampling` is not used by default, but you should probably use it if you are producing post-fit distributions. Without this the uncertainty will be calculated assuming no correlations between nuisance parameters - this would be extremely rare in an typical fit.
24 
25 The output root file will have the following directory structure:
26 
27  output.root:
28  bin_1_prefit/
29  TH1F data_obs
30  TH1F process_1
31  TH1F process_2
32  ...
33  TH1F process_i
34  TH1F TotalBkg
35  TH1F TotalSig
36  bin_1_postfit/
37  TH1F data_obs
38  TH1F process_1
39  TH1F process_2
40  ...
41  TH1F process_i
42  TH1F TotalBkg
43  TH1F TotalSig
44  ...
45  bin_i_prefit/
46  ...
47  bin_i_postfit/
48  ...
49 
50 \warning In the post-fit output the signal histograms are not scaled to the fitted signal strength, rather a signal strength r=1 is assumed. It follows that the uncertainty on the signal processes does not include the uncertainty on this signal strength parameter.
51 
52 \note This program only produces the histograms - you will need to provide your own plotting script!
