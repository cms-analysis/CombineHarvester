1 BSM Model independent Limits using MorphingHhh or MorphingAZh {#ModelIndepHhhAZh}
2 =================================================================
3 
4 The model independent limits for the H->hh and A->Zh analyses (HIG-14-034) are slightly simpler than for the MSSM H->tautau analysis, since only one signal process is considered and hence no profiling is required. This removes the need for any kind of physics model at the text2workspace step.
5 
6 Creating datacards {#Hhh-Azh-p1}
7 ========================
8 
9 The first step is to create the datacards, which will be used to produce the limit later on. To do this, go into the Folder CombineHarvester/Run1BSMComb/ and then execute MorphingHhh or MorphingAZh. All of the programs in the following steps also need to be executed from this folder. Also make sure that all the files have been compiled beforehand:
10 
11  cd CombineHarvester/Run1BSMComb/
12  MorphingAZh
13  MorphingHhh
14 
15 MorphingHhh.cpp and MorphingAZh.cpp are setup similarly to Example2 and MorphingMSSMUpdate as documented previously. No additional option -m MH is needed as in the case of the MSSMUpdate cards because there is only one Higgs boson considered anyway for these analyses.
16 
17 
18 Creating the workspace {#Hhh-Azh-p2}
19 =======================================
20 
21 Now that the datacards have been created, we can create a workspace as follows:
22 
23 text2workspace.py -b output/AZh_cards_nomodel/htt_cmb_AZh.txt -o output/AZh_cards_nomodel/htt_cmb_AZh.root --default-morphing shape2
24 text2workspace.py -b output/hhh_cards_nomodel/htt_cmb_Hhh.txt -o output/hhh_cards_nomodel/htt_cmb_Hhh.root --default-morphing shape2
25 
26 This creates a workspace based on the combined datacard.
27 
28 
29 Calculating values {#Hhh-Azh-p3}
30 ========================
31 
32 Now that we have created the workspace, we can now run combineTool.py to run the limit for a given masspoint:
33 
34 python ../CombineTools/scripts/combineTool.py -m 260 -M Asymptotic output/hhh_cards_nomodel/htt_cmb_Hhh.root --freezeNuisances mH --setPhysicsModelParameters mH=260
35 python ../CombineTools/scripts/combineTool.py -m 220 -M Asymptotic output/AZh_cards_nomodel/htt_cmb_AZh.root --freezeNuisances mA --setPhysicsModelParameters mA=220
36 
37 This will run the asymptotic limit calculation for mH=260 for the H->hh cards and mA=220 for the A->Zh cards. Note that it is necessary to use the combine options to set the masspoint and to freeze the parameter controlling it so that it is not floated in the fit.
