584  There is no unique way to define a TH2 binning given an arbitrary
585  TGraph2D, therefore this function supports multiple named methods:
586 
587  - `BinEdgeAligned` simply takes the sets of x- and y- values in the
588  TGraph2D and uses these as the bin edge arrays in the TH2. The
589  implication of this is that when filling the bin contents interpolation
590  will be required when evaluating the TGraph2D at the bin centres.
591  - `BinCenterAligned` will try to have the TGraph2D points at the bin
592  centers, but this will only work completely correctly when the input
593  point spacing is regular. The algorithm first identifies the bin width
594  as the smallest interval between points on each axis. The start
595  position of the TH2 axis is then defined as the lowest value in the
596  TGraph2D minus half this width, and the axis continues with regular
597  bins until the graph maximum is passed.
598 
599  Args:
600  graph (TGraph2D): Should have at least two unique x and y values,
601  otherwise we can't define any bins
602  method (str): The binning algorithm to use
603  force_x_width (bool): Override the derived x-axis bin width in the
604  CenterAligned method
605  force_y_width (bool): Override the derived y-axis bin width in the
606  CenterAligned method
607 
608  Raises:
609  RuntimeError: If the method name is not recognised
610 
611  Returns:
612  TH2F: The exact binning of the TH2F depends on the chosen method
613  """
614  x_vals = set()
615  y_vals = set()
616 
617  for i in xrange(graph.GetN()):
618  x_vals.add(graph.GetX()[i])
619  y_vals.add(graph.GetY()[i])
620 
621  x_vals = sorted(x_vals)
622  y_vals = sorted(y_vals)
623  if method == 'BinEdgeAligned':
624  h_proto = R.TH2F('prototype', '',
625  len(x_vals) - 1, array('d', x_vals),
626  len(y_vals) - 1, array('d', y_vals))
627  elif method == 'BinCenterAligned':
628  x_widths = []
629  y_widths = []
630  for i in xrange(1, len(x_vals)):
631  x_widths.append(x_vals[i] - x_vals[i - 1])
632  for i in xrange(1, len(y_vals)):
633  y_widths.append(y_vals[i] - y_vals[i - 1])
634  x_min = min(x_widths) if force_x_width is None else force_x_width
635  y_min = min(y_widths) if force_y_width is None else force_y_width
636  x_bins = int(((x_vals[-1] - (x_vals[0] - 0.5 * x_min)) / x_min) + 0.5)
637  y_bins = int(((y_vals[-1] - (y_vals[0] - 0.5 * y_min)) / y_min) + 0.5)
638  print '[TH2FromTGraph2D] x-axis binning: (%i, %g, %g)' % (x_bins, x_vals[0] - 0.5 * x_min, x_vals[0] - 0.5 * x_min + x_bins * x_min)
639  print '[TH2FromTGraph2D] y-axis binning: (%i, %g, %g)' % (y_bins, y_vals[0] - 0.5 * y_min, y_vals[0] - 0.5 * y_min + y_bins * y_min)
640  # Use a number slightly smaller than 0.49999 because the TGraph2D interpolation
641  # is fussy about evaluating on the boundary
642  h_proto = R.TH2F('prototype', '',
643  x_bins, x_vals[
644  0] - 0.49999 * x_min, x_vals[0] - 0.50001 * x_min + x_bins * x_min,
645  y_bins, y_vals[0] - 0.49999 * y_min, y_vals[0] - 0.50001 * y_min + y_bins * y_min)
646  else:
647  raise RuntimeError(
648  '[TH2FromTGraph2D] Method %s not supported' % method)
649  h_proto.SetDirectory(0)
